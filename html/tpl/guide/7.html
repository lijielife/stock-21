T+1/T+D实盘规则

T+1是专为超短线炒股高手提供操盘资金，在股市中每天都有很多短线机会，当机会出现需要狙击时，T+1将成为您短线炒股的神器。您只需预存少量的操盘保证金，就可以让炒股资金放大4倍，最高可获得10万元的配资资金，今买明卖的操盘模式让您迅速赚取高收益。

T+D实盘操作模式同样是为短线炒股高手准备的，股民可自由操作2-20个交易日的实盘 。申请日为T日，T+D日为结算日，结算日14：30分前需将所持股票卖出。

操盘须知

1、在操盘过程中，请注意操盘资金的变化

①当操盘资金低于警戒线时，只能卖出股票，无法买入股票。 

②当操盘资金低于平仓线时，易操盘将有权将您的股票强制全部卖出。（不保证卖出价格） （为了避免发生不必要的平仓，请及时追加操盘保证金，让操盘资金保持在警戒线以上）

2、在操盘过程中，当操盘资金低于平仓线时

①如未被及时平仓，请及时卖出股票，以免出现超额亏损；

 

②如未被及时平仓，并发生超额亏损，那么超额亏损部分资金将由操盘人自行承担。

盈亏演示

注：以下演示，实盘资金10万元、交易3天、交易1只股票，举例如下 （在实际操盘中，资金最高10万，交易最多20天，股票交易次数不受限制）

投资盈利演示

申请实盘资金10万元，预付操盘保证金20000元 
总操盘资金120000元，每天账户管理费150元
第1天买入股票120000元
第2天继续持有
第3天股票涨了5%，卖出股票，终止操盘
== 结算盈亏 ==
  股票盈利金：120000元 × 5% = 6000元 
账户管理费：150元 × 3天 = 450元 
操盘保证金：20000元 
您可以获得：6000元 - 450元 + 20000元 = 25550元 
投资收益率：( 6000元 - 450元 ) ÷20000元 = 27.7%

投资亏损演示

申请实盘资金10万元，预付操盘保证金20000元 
总操盘资金120000元，每天账户管理费150元
第1天买入股票120000元
第2天继续持有
第3天股票跌了2%，卖出股票，终止操盘
== 结算盈亏 == 
  股票亏损金：120000元 × 2% =2400元 
  账户管理费：150元 × 3天 = 450元 
  操盘保证金：20000元 
  您可以获得：20000元 - 2400元 - 450元 = 17150元 
  投资亏损率：( 20000元 - 17150元 ) ÷ 20000元 = 14.25%
